Highscore Mode explained:

  • You start with 10000 points in "Normal Mode" and with "allow one color to be used multiple times" checked
  • For each level you go up ("Hard", "Insane") you gain 1000 points
  • For each level you go down ("Easy") you loose 1000 points
  • For every minuted passed you loose 250 points
  • For every new stage you loose 500 points
  • If you uncheck "allow one color to be used multiple times" you loose 2000 points
